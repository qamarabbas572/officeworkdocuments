Alternate method to install PHP-Sodium extension using " phpbrew "
Using this method, we don't bother about manually editing the php.ini file: -



·  Select the relevant PHP version from "phpbrew list" and set for both cli and apache/apache2.

·  If libsodium is already installed at system level then simply run command: " phpbrew -vvv ext install libsodium " 

                                                    Otherwise,

·  Install libsodium first by downloading a version from " https://download.libsodium.org/libsodium/releases/ " and run below commands in the extracted directory (e.g libsodium-1.0.18.):-

./configure

make && make check

sudo make install 


·  After that install php-sodium extension using command:  " phpbrew -vvv ext install libsodium "

·  Once installation is successful, go to the path where "sodium.so" exists. ( e.g. /opt/phpbrew/php/php-7.4.20/lib/php/extensions/no-debug-non-zts-20180731 )

·  Finally link sodium.so with libsodium.so i.e. " ln -s sodium.so libsodium.so ".

 

You may verify extension using "php -m" and the info.php URL as well.



 REF:
https://doc.libsodium.org/installation#integrity-checking

SuiteCRM 8.x || Installation using the CLI
Hi,

Recently faced an issue, SuiteCRM 8.1 was generating random errors while proceeding installation on multiple web browsers. After providing admin, DB and URL details it was not updating the provided information. 
As a work around, performed installation using the command line and issue was resolved: -

1- Modify php.ini for CLI accordingly:
vi /etc/php/8.0/cli/php.ini
" upload_max_filesize = 200M "
​
2- Run below command on the root of extracted SuiteCRM directory after modifying the quoted values as per your requirement:
./bin/console suitecrm:app:install -u "admin" -p "@dmin111" -U "root" -P "123" -H "localhost" -N "suite_8" -S "http://localhost/suitecrm8" -d "no"
e.g 
./bin/console suitecrm:app:install -u "admin" -p "admin123" -U "root" -P "@UGARcrm00792" -H "localhost" -N "suite8" -S "https://plugins.rolustech.com:44324/" -d "yes"


Once installation is done, please open below link in the private browsing mode and try signing in.
http://localhost/suitecrm8/public/#/Login
or
https://localhost/suitecrm8/public/#/Login
( Note: In my case root directory for installation was /var/www/html/sugarcrm8, it can vary for others )

Install python dependencies on ubuntu
				#!/bin/bash

		# Script to install all Python dependencies using pip in Ubuntu

		# Check if pip is installed
		if ! command -v pip &> /dev/null; then
			echo "pip is not installed. Installing pip..."
			# Install pip using apt-get
			sudo apt-get update
			sudo apt-get install -y python3-pip
		fi

		# Specify the requirements file
		REQUIREMENTS_FILE="requirements.txt"

		# Check if requirements file exists
		if [ ! -f "$REQUIREMENTS_FILE" ]; then
			echo "No requirements file found. Please provide a requirements.txt file."
			exit 1
		fi

		# Install dependencies using pip
		echo "Installing dependencies from $REQUIREMENTS_FILE..."
		pip install -r "$REQUIREMENTS_FILE"

		# Output success message
		echo "All dependencies have been installed successfully."
